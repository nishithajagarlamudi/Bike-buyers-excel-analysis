# 🚲 Bike Sales Analysis — Excel Dashboard

## 📊 Project Overview
Analyzed a dataset of 174 customers to identify key factors 
that influence bike purchasing decisions using Excel.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers & Filters
- Charts & Data Visualization
- Data Cleaning & Transformation

## 📌 Key Business Questions Answered
- Which age group buys bikes the most?
- Does commute distance affect bike purchases?
- How does income vary between buyers and non-buyers?
- Does gender influence bike purchasing behavior?

## 🔍 Key Insights

### 1. Age Group
- Middle Age customers dominated purchases — 83 out of 94 
  total buyers
- Old age group had very low purchases — only 9 buyers
- Adolescents had zero purchases

### 2. Commute Distance
- Customers commuting 0-1 miles were the highest buyers — 
  50 purchases
- Purchases declined sharply as commute distance increased
- Customers commuting 10+ miles had the lowest purchases — 
  only 3 buyers

### 3. Income vs Gender
- Male bike buyers had average income of $68,049
- Female bike buyers had average income of $61,698
- Non-buyers had higher average incomes across both genders

### 4. Business Recommendation
Target marketing campaigns towards middle-aged customers 
with short commutes (0-1 miles) as they represent the 
highest conversion segment.
